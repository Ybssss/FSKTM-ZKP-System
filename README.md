# FSKTM-ZKP-System

**ZERO-KNOWLEDGE PROOF-BASED AUTHENTICATION FOR PASSWORDLESS LOGIN IN A WEB-BASED EVALUATION SYSTEM FOR UTHM FSKTM POSTGRADUATE RESEARCH SYMPOSIUM**

FSKTM-ZKP-System is a secure web-based evaluation platform developed for the UTHM Faculty of Computer Science and Information Technology postgraduate research symposium. The system supports postgraduate evaluation management, session scheduling, panel assignment, official evaluation reports, historical feedback access control, student material uploads, and attendance verification.

The system implements passwordless authentication using Zero-Knowledge Proof concepts, where users authenticate through registered cryptographic devices without submitting traditional passwords. It includes role-based access control for admins, panels, and postgraduate students.

## Key Features

- Zero-Knowledge Proof-based passwordless authentication
- Admin, panel, and postgraduate student role-based access control
- Postgraduate research symposium session scheduling
- Automatic evaluation generation after timetable scheduling
- Proposal Defense, Pre-Viva, and Progress Assessment evaluation support
- AI-assisted panel expertise matching
- Historical Feedback Vault with access request and approval workflow
- Student supporting material upload for evaluation sessions
- Official evaluation report viewing
- Attendance verification using QR/PIN gateway
- Secure permission withdrawal for approved historical access
